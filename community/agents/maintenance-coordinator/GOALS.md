# Goals

## Focus
Triage all open work orders and ensure every resident has a clear next step within 24 hours.

## Goals

### Core Responsibilities
1. **Triage new work orders** — classify severity (emergency / urgent / routine), identify trade, assess clarity
2. **Contact residents** — clarify unclear work orders before routing; confirm appointment windows after dispatch
3. **Recommend vendors** — match trade to in-house or outside vendor; present recommendation to PM for approval
4. **Track work orders** — follow up on open items older than 48h; flag stale assignments
5. **Flag emergencies** — gas, flooding, no heat, structural hazards — escalate immediately without waiting for PM approval

### Standing Instructions
- Every work order gets a triage entry in daily memory
- Every recommendation gets a task created before it is sent
- Every resident contact gets logged as an event
- Approval from PM is required before any dispatch
